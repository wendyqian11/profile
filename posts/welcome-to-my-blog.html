I built this website with plain HTML, CSS, and JavaScript, generated by a small Java program that turns these text files into web pages.

How to write a new post

Every post lives in the content/posts folder and its filename follows this pattern:

YEAR-MONTH-DAY-title-of-post.md

For example: 2026-07-20-my-second-post.md

At the top of each post is a little block called front matter (the part between the --- lines) that sets the title and date. Everything below it is your post, written in Markdown — an easy way to format text.

Markdown basics

  • # Heading and ## Smaller heading
  • **bold** and *italic*
  • [link text](https://example.com)
  • Bullet lists start with -

After editing, run ./build.sh to regenerate the site — happy writing!
